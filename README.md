### QA
1. 如果评分模型升级，如何避免评分标准漂移？
    - 使用固定的数据集测试
    - 模型升级时评估阶段先使用新模型与旧方案同步对比结果
2. 如何做自动回归测试？
    - 单元测试
    - 使用固定的测试内容，验证结果稳定性
    - 使用CI/CD，提交代码后自动运行测试
3. 如果每天 2 万场面试，如何扩展系统？
    - 可以使用微服务架构，或者k8s等使用水平扩容
    - 异步处理如面试计划，评估报告等
    - 优化缓存设计
    - 部署多个相同模型来负载均衡
    - 大模型批处理
4. 如何防止 Prompt Injection？
    - 关键字过滤
    - 使用已有的一些防护软件如llama guard
    - 可以使用系统向量，不直接存储system—_prompt
5. 如何记录和审计完整面试过程？
    - 日志
    - 存储视频
    - 视频转文字对话存储
6. 如何避免模型被候选人“话术误导”？
    - system_prompt要求识别过滤话术误导
    - 人工审查
